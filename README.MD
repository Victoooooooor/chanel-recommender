# README

## Plateforme de recommendation de produit chanel selon la description, un image ou bien les deux

# 1. Créer un environnement virtuel (isole les dépendances du projet)
python -m venv venv

# 2. Activer l'environnement (obligatoire avant d'installer ou lancer)
venv\scripts\activate.ps1

# 3. Installer les librairies nécessaires
pip install -r requirements.txt

# 4. Lancer l'application Streamlit
python src/app/streamlit_app.py
